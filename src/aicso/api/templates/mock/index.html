ECS格式说明

生成的告警遵循 Elastic Common Schema (ECS) 8.x 标准：

  • event.* — 事件分类、动作、结果
  • rule.* — 规则ID、名称、严重级别、风险评分
  • source/destination.* — 网络五元组
  • threat.* — MITRE ATT&CK 战术和技术映射
  • host/user/agent.* — 主机、用户、Agent信息
  • file/process.* — 文件和进程信息（恶意软件场景）

数据自动转换为 AiCSO Alert 模型，保留完整 ECS 原始数据在 raw_log 字段中。
